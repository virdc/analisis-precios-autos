# Base de precios de autos de Argentina

## Dataset
- https://www.kaggle.com/datasets/ahmettalhabektas/argentina-car-prices/data

## Dependencias
- Python 3.13.5
- Pandas
- Matplotlib
- Seaborn
- Numpy

## Install
Ejecutar este comando 
```bash
pip install -r requirements.txt
```
